# sistem-arsip-kima

Project ini sudah direfactor menjadi 2 bagian:

- `frontend/` : aplikasi React + Vite + Tailwind
- `backend/` : API server Node.js sederhana

## Menjalankan Frontend

```bash
cd frontend
npm run dev
```

## Menjalankan Backend

```bash
cd backend
npm run dev
```

Backend menyediakan health check di:

- `GET /api/health`
